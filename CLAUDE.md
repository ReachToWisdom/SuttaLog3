# SuttaLog3 — 경전 중심 빠알리어 학습앱

> **"문법책이 아니라, 경전이 교재다"** · 수행을 위한 빠알리어

## 기술 스택

- React 19 + TypeScript + Vite 8 + Tailwind CSS 4
- Firebase (Firestore + Google Auth) / GitHub Pages 자동 배포
- PWA (Network First — 온라인 최신, 오프라인 캐시)
- 포트: 3024 (dev) / 배포: reachtowisdom.github.io/SuttaLog3/

## 학습 경로

```
자모 발음 → 기초 문법(L2~6)
  → 행복경 → [분기: 보배경/자비경 선택]
  → 추가학습1(대명사/분사) → 전법륜경(15단락 전문)
  → 추가학습2(미래형/수동) → 무아경(14단락 전문)
  → 사념처경(39단락 전문)
  → 추가학습3(복합어/연성법/총정리)
```

## 경전 전문 수록 현황

| 경전 | 단락 | 상태 |
|---|---|---|
| 행복경 Maṅgala | 13게송 | 전문 |
| 보배경 Ratana | 17게송 | 전문 |
| 자비경 Metta | 10게송 | 전문 |
| 전법륜경 Dhammacakka | 15단락 | 전문 (삼전십이행상 포함) |
| 무아경 Anattalakkhaṇa | 14단락 | 전문 (오온 5×반복 포함) |
| 사념처경 Mahāsatipaṭṭhāna | 39단락 | 전문 (신/수/심/법+팔정도 상세) |

## 핵심 규칙

1. **수행 목적**: 학문적 분석/용어 퀴즈 금지, "뜻 이해"에만 집중
2. **경전 전문**: 생략/발췌 금지, 모든 단어 WORDS에 빠짐없이
3. **발음 1:1**: 원문 각 단어 바로 아래 한글 발음, ON/OFF 가능
4. **퀴즈 랜덤**: 매번 다른 순서, 격변화 함정 포함
5. **SSOT**: 설정 config, 명구 quotes.ts, 발음 pron-display.ts
6. **이어하기**: 스텝 진도 자동 저장, 범위 초과 시 리셋
7. **텍스트 셀렉트 금지**: user-select: none
8. **테이블 가로 스크롤**: overflow-x-auto (모바일 잘림 방지)
