# SuttaLog3 — 경전 중심 빠알리어 학습앱

> **"문법책이 아니라, 경전이 교재다"**

## 학습 철학

1. **경전 원문**으로 배운다 (문법책 아님)
2. **발음 → 문법 → 독해** 순서 (기초부터 쌓기)
3. **에빙하우스 3회 반복** — 새 것을 배울 때 이전 것이 복습됨
4. 음성학 이론 없이 **"이 글자는 이렇게 읽는다"** 에 집중
5. 대충 알면 못 맞추게 — **격변화 함정 선택지**

## 학습 경로

```
Part 0: 자모 발음 (모음 8 + 자음 33)
        ↓
기초 문법 개론 (3성 · 8격 · 격변화표 · 동사 전체 지도)
        ↓
Part 1: 행복경 Maṅgala Sutta (연성법 16가지 + 기초 단어)
        ↓
   [분기점] ─── 기초 더 다지기? → 보배경 Ratana + 자비경 Metta
        ↓
심화 문법 (i/u-어간 · 과거형 · 분사 · 복합어)
        ↓
Part 2: 전법륜경 (사성제 · 팔정도 · 문법 시작)
        ↓
        무아경 (오온 · 격변화 심화)
        ↓
        사념처경 (수행 정형문 · 긴 텍스트)
```

| 타입 | 용도 | 예시 |

## 퀴즈 설계

### 함정 선택지 (대충 알면 못 맞추게)
- **격변화**: me(주격) vs mayā(구격) vs mama(속격)
- **접미어**: sutaṃ(대격) vs sutassa(속격) vs sutena(구격)
- **연성법**: maṅgalaṃ vs maṅgalamuttamaṃ (병합)
- **유사 철자**: sutta(경전) vs suttaṃ vs sutaṃ(들은 것)

### 모든 단어 커버
- 본문 모든 단어가 퀴즈에 최소 1번 등장
- 매번 학습 시 순서 랜덤 셔플

### 해설 포함
- 정답/오답 후 즉시 해설
- 상세 문법 설명 + 💡 팁
- 이전 단원 연결 ("행복경에서도 나왔습니다")

## 화면 구성 (5탭)

| 탭 | 내용 |
|---|---|
| 🏠 홈 | 히어로 카드 + 통계 + 명구 (단어별 1:1 발음) |
| 📚 목차 | 경전 경로 / 문법 경로 탭 전환 |
| 🔄 복습 | 오답복습 / 랜덤퀴즈 / 약한단어 / 단원리뷰 |
| 📊 기록 | 7일 차트 + 평균 + 히트맵 + 약한단어 TOP |
| ⚙️ 설정 | 소리 / 발음표시 / 영어발음병기 / 동기화 |

## 게임화

| 요소 | 설명 |
|------|------|
| 하트(연꽃) 3개 | 오답 시 감소, 0이면 처음부터 |
| 진도 바 + 스텝 번호 | 상단 표시 |
| 건너뛰기 | 하트 감소 없이 다음으로 |
| 이어하기 | 마지막 스텝부터 재시작 |
| 되돌아가기 | 이전 스텝으로 |
| 완료 화면 | 정답수 + 하트 + 시간 + 정확도% |

## 발음 표시

```
원문 각 단어 바로 아래 1:1 매칭:
  Evaṃ    me    sutaṃ
  에왕     메    수땅

설정 ON/OFF:
- 발음 표시: ON(기본) → 원문 밑에 한글 발음
- 영어 발음 병기: OFF(기본) → ON 시 "에왕 (eh-wang)"
```
