# SuttaLog3 — 경전 중심 빠알리어 학습앱

> **"문법책이 아니라, 경전이 교재다"**

---

## 학습 철학

1. **경전 원문**으로 배운다 (문법은 경전 맥락에서)
2. **기초 문법 지도 → 경전 실전 → 심화 문법** (전체 구조 파악 후 실전)
3. **문법이 주, 단어는 보조** — 단어는 자율 열람 + 카운팅 복습
4. 새로 나오는 것만 설명, 배운 건 패스

---

## 학습 경로

```
Part 0: 자모 발음 (모음 8 + 자음 33)
   ↓
기초 문법 개론 (3성 · 8격 · a-어간 격변화 · 동사 현재형 · 연성법)
   ↓
Part 1: 게송 경전 — 행복경 (발음+연성법 실전)
   ├── [선택] 보배경, 자비경 (기초 보강)
   ↓
심화 문법 (i/u-어간 · 과거형 · 분사 · 복합어)
   ↓
Part 2: 산문 경전 — 전법륜경 → 무아경 → 사념처경 (독해)
```

## 단원 구성 (각 경전별)

### 구조: intro → 단어목록 → 문법 → 본문 → 퀴즈

| 순서 | 스텝 타입 | 설명 |
|------|----------|------|
| 1 | `intro` | 경전 소개 (제목, 출처, 학습 포인트) |
| 2 | `vocab-list` | **핵심 단어 한 화면** — 탭하여 자율 학습, 열람 카운팅 |
| 3 | `teach-grammar` | **문법 설명 (메인!)** — 예시+테이블+팁 |
| 4 | `verse` | **경전 원문** — 문법 글로서리 기본 표시 |
| 5 | 퀴즈 | 게송: 단어 퀴즈 / 산문: **문법 퀴즈** |

### 단어 학습 방식 (변경)

- ❌ 단어 하나씩 카드로 넘기며 학습 (폐기)
- ✅ **vocab-list**: 전체 단어를 한 화면에 보여주고 탭으로 자율 학습
- ✅ **verse 글로서리**: 본문 아래에 단어+문법 기본 표시
- ✅ **열람 카운팅**: 탭한 단어를 추적하여 복습 대상으로 활용

### 본문(verse) 표시 방식

```
┌─────────────────────────────────┐
│         ✦ PĀḶI ✦               │
│  Ekaṃ  samayaṃ  bhagavā  ...   │  ← 빠알리 원문 (단어별)
│  에깡   사마양    바가와-   ...   │  ← 발음 (ON/OFF)
└─────────────────────────────────┘

한 때 세존께서 바라나시에서 머무셨다.   ← 한글 번역

📝 문법 설명 (기본 표시, 접기/펼치기)
┌─────────────────────────────────┐
│ Ekaṃ — 하나의 (수사 중성 대격)    │
│ samayaṃ — 때 (남성 단수 대격)     │
│ bhagavā — 세존 (남성 단수 주격)   │
│ viharati — 머무시다               │
│   (현재 3인칭 단수, vi + harati)  │
└─────────────────────────────────┘

[🔊 전체 듣기]  [다음 →]
```

### 퀴즈 유형

| 경전 종류 | 퀴즈 내용 | 예시 |
|----------|----------|------|
| 기초 (자모) | 발음 퀴즈 | "ca의 발음은?" |
| 게송 (행복경 등) | 단어 뜻 + 역방향 | "bhagavā의 뜻은?" |
| 문법 단원 | 문법 이해도 | "a-어간 남성 처격 어미는?" |
| 산문 경전 | **문법 퀴즈** | "viharati의 문법적 분해는?" |

## 복습 시스템

- **단어 열람 카운팅**: vocab-list/verse에서 탭한 단어 추적
- **복습 탭**: 카운팅된 단어 기반으로 복습 퀴즈 제공
- **에빙하우스 3회 반복**: 퀴즈에서 틀린 단어 우선 복습

## UI 구성

### 하단 네비게이션
| 탭 | 기능 |
|----|------|
| 🏠 홈 | 일일 명구 + 학습 현황 |
| 📚 목차 | 경전 경로 / 문법 경로 (탭 전환) |
| 🔄 복습 | 틀린 문제 + 열람 단어 복습 |
| 📊 통계 | 학습 시간 · 연속 학습 |

### 학습 화면
- **프로그레스 바** + 하트 3개 + 발음 설정 토글(⚙)
- **이전/다음 버튼**: 모든 비퀴즈 스텝에 제공
- **건너뛰기**: 퀴즈에서 하트 감소 없이 넘기기
