# SuttaLog3 — 경전 중심 빠알리어 학습앱

> **"문법책이 아니라, 경전이 교재다"**

---

## 학습법

### 원칙

1. **경전 원문**으로만 배운다 (문법책 없음)
2. **발음 → 연성법 → 문법** 순서 (기초부터 쌓기)
3. **에빙하우스 3회 반복** — 새 것을 배울 때 이전 것이 복습됨
4. 새로 나오는 것만 설명, 배운 건 패스
5. 3회 반복 완료 후 자동 OFF (독해 집중), ON/OFF 토글 제어

### 핵심 구조

```
새 항목을 배우는 동안, 이전 항목이 자동 복습된다

모음 학습 중   → (복습 대상 없음)
자음 학습 중   → 모음 복습
연성법 학습 중 → 자음 복습
문법 학습 중   → 연성법 복습
독해 중       → 모든 것이 자연 반복
```

### 학습 구간

| 구간 | 1회차 (새로 배움) | 복습 (이전 것) |
|------|------------------|---------------|
| **1** | 모음 8자 | — |
| **2** | 자음 33자 | 모음 2·3회 → **모음 완료** |
| **3** | 연성법 16가지 | 자음 2·3회 → **자음 완료** |
| **4** | 문법 (격변화, 동사활용) | 연성법 2·3회 → **연성법 완료** → 자동 OFF |
| **5** | 독해 | 문법 자연 반복 |

## 경전 순서

### 기초 경전: Maṅgala Sutta (행복경)

발음·자음·연성법을 배우기 위한 **기초 교재**로 사용.

행복경을 마친 뒤, 본격적인 경전 학습(독해·문법)으로 진입.

### 본 학습 경전 순서

행복경 이후, 아래 순서로 진행:

| 순서 | 경전 | 학습 초점 |
|------|------|----------|
| 1 | Maṅgala Sutta (행복경) | 발음 → 자음 → 연성법 (기초) |
| 2 | Dhammacakkappavattana Sutta (전법륜경) | 문법 시작 + 독해 입문 |
| 3 | Anattalakkhaṇa Sutta (무아경) | 독해 심화 |
| 4 | Mahāsatipaṭṭhāna Sutta 발췌 (팔정도경/사념처경) | 독해 심화 |
| 5+ | 추가 경전 (법구경, 자비경 등) | 독해 확장 |

## 경전별 성격 비교

### 1. Maṅgala Sutta (행복경) — 기초 교재

- **출처**: 소부(Khuddakapāṭha) + 숫따니빠따(Suttanipāta)
- **형식**: 운문 (게송 12개, 각 2행)
- **내용**: 천신이 "최상의 행복이 무엇입니까?" 여쭈자, 부처님이 38가지 행복을 답하심
- **난이도**: ★☆☆☆☆ (가장 쉬움)
- **특징**:
  - 짧은 게송, 반복 구문 ("etaṃ maṅgalamuttamaṃ")
  - 일상 윤리 중심 (전문 용어 적음)
  - 남방불교에서 가장 자주 암송하는 경전
  - 자모 41자 중 30자, 연성법 16가지 전부 커버 가능
- **이 앱에서의 역할**: 발음·자음·연성법 기초를 다지는 교재

```
Bahū devā manussā ca, maṅgalāni acintayuṃ,
ākaṅkhamānā sotthānaṃ, brūhi maṅgalamuttamaṃ.

많은 천신과 인간들이 행복을 생각하며 안녕을 바라나니,
최상의 행복을 말씀해 주소서.
```

### 2. Dhammacakkappavattana Sutta (전법륜경) — 첫 번째 독해

- **출처**: 상윳따 니까야(SN 56.11)
- **형식**: 산문 + 운문 혼합
- **내용**: 부처님의 **첫 설법**. 다섯 비구에게 사성제와 팔정도, 중도를 설하심
- **난이도**: ★★★☆☆ (중간)
- **특징**:
  - 불교 교학의 핵심 (사성제·팔정도·중도)
  - 반복 구문 풍부 ("idaṃ dukkhaṃ ariyasaccaṃ")
  - 산문 입문에 적합
  - 격변화·동사활용이 체계적으로 등장
- **이 앱에서의 역할**: 문법 학습 시작 + 첫 독해 경험

```
Ekaṃ samayaṃ bhagavā bārāṇasiyaṃ viharati isipatane migadāye.

한 때 세존께서 바라나시의 이시빠따나 녹야원에 머무셨다.
```

### 3. Anattalakkhaṇa Sutta (무아경) — 독해 심화

- **출처**: 상윳따 니까야(SN 22.59)
- **형식**: 산문
- **내용**: 부처님의 **두 번째 설법**. 오온(색·수·상·행·식)이 무아임을 설하심
- **난이도**: ★★★★☆ (어려움)
- **특징**:
  - 오온 분석의 반복 구조 ("rūpaṃ anattā... vedanā anattā...")
  - 격변화 심화 (여러 격이 한 문장에 혼합)
  - 논리적 논증 구조
  - 부처님의 첫 5제자가 이 경을 듣고 아라한과를 증득
- **이 앱에서의 역할**: 격변화 심화 학습 + 산문 독해력 강화

```
Rūpaṃ bhikkhave anattā. Rūpañca hidaṃ bhikkhave attā abhavissa,
nayidaṃ rūpaṃ ābādhāya saṃvatteyya.

비구들이여, 물질은 자아가 아니다. 만약 물질이 자아라면,
물질이 병들지 않아야 할 것이다.
```

### 4. Mahāsatipaṭṭhāna Sutta 발췌 (사념처경) — 수행 안내서

- **출처**: 디가 니까야(DN 22) / 맛지마 니까야(MN 10)
- **형식**: 산문 (정형문 반복)
- **내용**: 사념처(몸·느낌·마음·법) 수행법을 상세히 설하심. 팔정도 부분 포함
- **난이도**: ★★★★★ (가장 어려움)
- **특징**:
  - 가장 긴 경전 중 하나 (전체는 발췌하여 학습)
  - 정형 반복문 ("iti ajjhattaṃ vā... bahiddhā vā...")
  - 수행 실천 지침 (관찰·알아차림)
  - 팔정도·사성제 내용이 경전 후반부에 상세 설명
- **이 앱에서의 역할**: 긴 텍스트 독해 도전 + 수행 용어 학습

```
Ekāyano ayaṃ bhikkhave maggo sattānaṃ visuddhiyā...

비구들이여, 이것은 중생의 청정을 위한 유일한 길이니...
```

## 왜 이 순서인가?

### 행복경이 먼저인 이유
- 짧은 게송 12개로 **발음·자음·연성법**을 전부 배울 수 있음
- 일상 윤리 내용이라 전문 용어 부담 없음
- 기초를 다진 후 어려운 경전으로 진입

### 전법륜경 → 무아경 → 사념처경 순서인 이유
- **역사적 순서**: 부처님이 실제로 설하신 순서 (첫 설법 → 두 번째 설법)
- **교학적 순서**: 사성제 → 무아 → 수행법 (이해 → 통찰 → 실천)
- **난이도 순서**: 중간 → 어려움 → 최고난도
- De Silva Pali Primer의 문법을 경전 맥락에서 자연스럽게 습득

## 각 단원 학습 구성

```
1. 경전 원문 제시 (빠알리 + 한글 번역 + 음성)
2. 단어 분해 (어근, 접두사/접미사, 격변화, 동사활용)
3. 문장 구조 분석 (주어/동사/목적어, 격의 역할)
4. 문법 포인트 (처음 만나는 문법 + 이전 문법 복습 연결)
5. 연습 문제 (퀴즈, 듣기, 배열, 필기)
6. 암송 (원문 표시 + 빈칸 채우기)
```
