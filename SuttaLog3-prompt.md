# SuttaLog3 — 경전 중심 빠알리어 학습앱 구현 프롬프트

> **"문법책이 아니라, 경전이 교재다"** · 수행을 위한 빠알리어

---

## 1. 프로젝트 개요

빠알리어 문법 교재(De Silva Pali Primer, 32과)를 뼈대로 하되, 각 과의 예문과 연습을 **실제 경전 원문**으로 대체하는 학습앱.

**핵심 원칙**:
- 교재 각 과를 배우면서, **해당 문법이 실제로 나오는 경전 구절**을 바로 학습
- 한 화면에 한 개념 (스마트폰 최적화)
- 학문적 분석/용어 암기 금지 — **"이 문장의 뜻은?"** 에만 집중
- 경전 전문 수록 (생략/발췌 금지)

## 3. 학습 경로 (교재 32과 + 경전 5개 통합)

### 한 과의 학습 흐름 (핵심!)

각 과는 **짧은 화면 여러 개**로 구성:
```
[화면1] 문법 규칙 (1~2문장 + 어미)
[화면2] 교재 예문 (2~3개)
[화면3] 경전 실전 — 같은 문법이 나오는 경전 구절 + 1:1 발음
[화면4] 퀴즈 (격변화 함정 포함)
```

### 교재 32과와 경전 배치

```
1과: 자모와 발음 (모음 8 + 자음 33)
  → 예시: 행복경 단어 (자연스러운 것만, 억지 금지)

2과: 목적격 (-ṃ/-e)
  → 경전: "Ekaṃ samayaṃ" "dhammaṃ" (행복경)
3과: 구격 (-ena/-ehi)
  → 경전: "lokadhammehi" (행복경)
4과: 여격/탈격
5과: 소유격
  → 경전: "Anāthapiṇḍikassa" (행복경)
6과: 처소격
  → 경전: "Sāvatthiyaṃ viharati" "ārāme" (행복경)
7과: 호격
  → 경전: "bhikkhave" (전법륜경/무아경)
8과: 전체 정리

9과: 절대분사 (-tvā)
  → 경전: "katvāna" (행복경), "ābhujitvā" (사념처경)
10과: 부정사 (-tuṃ)
  → 경전: "samanupassituṃ" (무아경)
11과: 중성명사
  → 경전: "dukkhaṃ" "rūpaṃ" (전법륜경/무아경)
12과: 동사 현재형
  → 경전: "viharati" "gacchanti" (행복경)
13과: 특수 동사 + -nā어간
  → 경전: "pajānāti" (사념처경)

★ 행복경 전문 (13게송)
  → 연성법 16가지 함께 학습

14과: 미래형
15과: 원망형 (-eyya)
  → 경전: "bhāvaye" (자비경), "abhavissa" (무아경)
16과: 명령형 (-tu/-hi)
  → 경전: "brūhi" (행복경), "suṇantu" (보배경)
17과: 과거형 + 어두첨자 a-
  → 경전: "acintayuṃ" (행복경), "udapādi" (전법륜경)
18과: ā-여성명사
  → 경전: "vedanā" "paññā" (무아경)

★ 보배경 (17게송) — 선택
★ 자비경 (10게송) — 선택

19과: 과거분사
  → 경전: "sutaṃ" "abhisambuddhā" (행복경/전법륜경)
20과: i-여성명사 + ī-여성명사
  → 경전: "jāti" (사념처경 고성제)
21과: 현재분사
  → 경전: "ākaṅkhamānā" (행복경), "kāyānupassī" (사념처경)
22과: 미래수동분사
  → 경전: "sevitabbā" "pariññeyyaṃ" (전법륜경)

★ 전법륜경 전문 (15단락)

23과: 사역형
24과: u-여성명사
25과: i-남성명사
26과: ī-남성명사
27과: u-남성명사 + ū-남성명사
28과: u/ar-어간 + 친척명사
29과: i-중성 + u-중성명사

★ 무아경 전문 (14단락)

30과: -vantu/-mantu 형용사
  → 경전: "bhagavā" "satimā" (행복경/사념처경)
31과: 대명사 (인칭/지시)
32과: 대명사 (관계/의문) + 대명사적 형용사
복합어 · 접두사 · 접미사
수사 · 비교급 · 최상급
연성법 상세 · 총정리

★ 사념처경 1부 — 신념처 (16단락)
★ 사념처경 2부 — 수심법념처+팔정도 (23단락)
```

## 4. 화면 설계 — 한 화면에 한 개념

### 문법 규칙 화면 (짧게!)
```
┌──────────────────────┐
│ 📐 2과: 목적격        │
│                      │
│ ~을/를 에 해당합니다   │
│ 단수: -ṃ  복수: -e   │
│                      │
│  dhammaṃ  🔊         │
│  담마ṃ               │ ← 발음 (ON/OFF)
│  법을                 │
│                      │
│         [다음 →]      │
└──────────────────────┘
```

### 경전 실전 화면
```
┌──────────────────────┐
│ 🪷 행복경에서 목적격   │
│      ✦ PĀḶI ✦       │
│  Evaṃ    me    sutaṃ │
│  에왕     메    수땅  │ ← 1:1 발음
│                      │
│ "이와 같이 나는 들었다"│
│                      │
│ 📝 sutaṃ = 들은 것    │
│    (중성 주격 -ṃ)     │
│                      │
│ [🔊 듣기] [다음 →]    │
└──────────────────────┘
```

### 퀴즈 (격변화 함정)
```
┌──────────────────────┐
│ "dhammaṃ"의 뜻은?     │
│                      │
│ A  법을      ← 정답   │
│ B  법이      ← 주격   │
│ C  법의      ← 소유격  │
│ D  법에서    ← 처소격  │
│                      │
│    [건너뛰기 →]       │
└──────────────────────┘
```

## 7. 퀴즈 설계

### 유형 (학문적 분석 없이 뜻 이해 집중)
1. 뜻 퀴즈: "이 빠알리의 뜻은?"
2. 역방향: "이 뜻에 해당하는 빠알리는?" (격변화 함정)
3. 빈칸 채우기: 경전 문장에서 올바른 격 형태 선택
4. 문장 조립: 한글 → 빠알리 조각 (짧은 구절만 3~8단어)
5. 쌍 비교: "niccaṃ과 aniccaṃ의 차이는?"
6. 발음: 글자→발음 / 발음→글자

### 문제은행: 경전당 40~78문제 (총 299문제), 매번 랜덤 10개
